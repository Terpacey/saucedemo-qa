| Status | Open |
| Severity | Critical |
| Priority | P0 |
| Defect Description | On the checkout step 1 page for problem_user, input directed at the Last Name field is incorrectly written to the First Name field. The Last Name field does not accept input. A single keypress overwrites the First Name value with the last character typed; a pasted or autofilled value overwrites it with the full string. As a result, the Last Name field remains empty and the user cannot proceed past step 1 |
| Expected Result | The Last Name field accepts input independently; the First Name field is unaffected by input in the Last Name field |
| Actual Result | Typing or pasting into the Last Name field overwrites the First Name field; the Last Name field remains empty; clicking Continue produces "Error: Last Name is required"; checkout cannot be completed |
| Steps to Reproduce | 1. Navigate to https://www.saucedemo.com<br>2. Log in as problem_user (password: secret_sauce)<br>3. Add Sauce Labs Backpack to the cart<br>4. Navigate to the cart page and click Checkout<br>5. Enter a value in First Name<br>6. Attempt to enter a value in Last Name — observe that the First Name field is overwritten and Last Name remains empty<br>7. Click Continue — "Error: Last Name is required" is displayed |
