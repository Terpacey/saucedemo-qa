| Defect ID | D_016 |
| Status | Open |
| Severity | Critical |
| Priority | P0 |
| Defect Description | Cart contents persist across user sessions. When a user logs out and a different user logs in, the new session inherits the previous user's cart state |
| Expected Result | Each user session begins with an empty cart. Cart contents should not carry over between different user accounts |
| Actual Result | The cart retains items added in a previous user's session when a new user logs in |
| Steps to Reproduce | 1. Navigate to https://www.saucedemo.com<br>2. Log in as standard_user (password: secret_sauce)<br>3. Add one or more items to the cart<br>4. Log out via the burger menu<br>5. Log in as a different user (e.g. problem_user)<br>6. Observe that the cart still contains the items from the previous session |
| Date Raised | 2026-03-28 |
| Reference | Exploratory observation during multi-user testing |
